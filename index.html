QuickDraw-LITE

QuickDraw-LITE is an implemenatation of Google QuickDraw that uses CNN to recognize various drawings. The CNN was trained to recognize 150 classes using the quick draw dataset. It will display the top five predictions for any drawing. The original QuickDraw game is trained for 345 classes and the whole dataset can be found at QuickDraw dataset
